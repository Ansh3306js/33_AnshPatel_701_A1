# Problem 1: AJAX Hello Server

A simple Node.js application using Express that serves a GET route `/gethello` returning plain text "Hello NodeJS!!", and a webpage that makes an AJAX request via Fetch API to display it dynamically.

## How to Run

1. Open your terminal and change directory to this folder:
   ```bash
   cd "01_hello_ajax"
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Start the server:
   ```bash
   npm start
   ```

4. Open your browser and navigate to:
   ```
   http://localhost:3000
   ```

5. Click the "Fetch Greeting" button to fetch and display the greeting.
