# Problem 3: Chatbot CLI Module

A reusable rule-based chatbot module representing a college helpdesk, used in an interactive command-line application.

## How to Run

1. Open your terminal and change directory to this folder:
   ```bash
   cd "03_chatbot_module"
   ```

2. Run the application (no external npm dependencies are needed, as it relies on built-in Node modules):
   ```bash
   node app.js
   ```

3. Type queries like:
   - "How do I apply for admission?"
   - "What are the tuition fees?"
   - "When are the exam dates?"
   - "exit"
