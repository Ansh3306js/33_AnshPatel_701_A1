# Problem 5: Zip Extraction

A Node.js program that extracts a `.zip` archive into a directory using the `adm-zip` package.

## How to Run

1. Open your terminal and change directory to this folder:
   ```bash
   cd "05_extract_zip"
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Run the extraction script:
   ```bash
   npm start
   ```

4. The script will dynamically generate a `sample.zip` file, extract it to `extracted_folder`, and list the extracted files in the console.
