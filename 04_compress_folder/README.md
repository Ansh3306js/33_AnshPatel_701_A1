# Problem 4: Folder Compression

A Node.js program that compresses a folder into a `.zip` archive using the `archiver` package.

## How to Run

1. Open your terminal and change directory to this folder:
   ```bash
   cd "04_compress_folder"
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Run the compression script:
   ```bash
   npm start
   ```

4. The script will automatically create a directory named `source_folder` with sample text files, and compress it to `output.zip`. You can check the terminal output for the final archive size.
