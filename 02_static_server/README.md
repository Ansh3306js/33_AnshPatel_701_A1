# Problem 2: Static Resource Server

A web server built with Express that serves static resources (HTML, CSS, JS, and images) from a `public` folder using `express.static` middleware.

## How to Run

1. Open your terminal and change directory to this folder:
   ```bash
   cd "02_static_server"
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Start the server:
   ```bash
   npm start
   ```

4. Open your browser and navigate to:
   ```
   http://localhost:3000
   ```

5. The page, stylesheet, client-side script, and banner image will load statically from the server. Click the button to test the static JavaScript asset execution.
