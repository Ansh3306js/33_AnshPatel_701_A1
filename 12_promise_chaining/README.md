# Problem 12: Promise Chaining

A Node.js program illustrating how to run three dependent asynchronous tasks sequentially using Promise constructors and `.then()` chaining, returning the results of each task to the subsequent call.

## How to Run

1. Open your terminal and change directory to this folder:
   ```bash
   cd "12_promise_chaining"
   ```

2. Start the script:
   ```bash
   npm start
   ```

3. Observe the sequential 1-second delays corresponding to each task running in order, showing how outputs propagate through the `.then()` chain.
