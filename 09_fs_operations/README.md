# Problem 9: File System Operations

A Node.js console application demonstrating at least 5-6 core functions from the native `fs` module with descriptive output for each step.

## How to Run

1. Open your terminal and change directory to this folder:
   ```bash
   cd "09_fs_operations"
   ```

2. Run the script:
   ```bash
   npm start
   ```

3. View the terminal output, which traces the creation, modification, reading, metadata querying, and renaming of files on the disk.
